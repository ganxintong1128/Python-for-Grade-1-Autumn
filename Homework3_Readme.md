# 个人作业3 技术报告

## 前置工具函数
1. **str2list()**
   - 功能：将字符串形式的大数转换为整型数码的列表
   - 细节：
     - 分别考虑符号和数值，返回一个（列表，整型）的元组（列表对应数值，整型对应符号：+1 和 -1）
     - 根据题目要求，在转换的同时完成列表的逆序排列

2. **list2str()**
   - 功能：将整型数码的列表逆序排列后转换为字符串
   - 细节：
     - 如果列表为空，返回 "0"
     - 去除高位的 0
     - 还原符号

3. **compare_list()**
   - 功能：比较两个列表

---

## 对列表进行的操作
1. **add_list()**
   - 细节：
     - carry 完成进位，先加再进
     - 高位补零

2. **sub_list()**
   - 细节：
     - borrow 实现借位，先借再减
     - 去除高位 0

3. **mul_list()**
   - 细节：
     - 默认将长数作为 lst1
     - carry 完成进位
     - 去掉高位 0

4. **div_list()**
   - 功能：实现列表形式的除法运算
   - 边界处理：
     - 除数为 0：抛异常
     - 被除数 < 除数：商为 0，余数为被除数本身
   - 输出：元组（商，余数），商和余数均为逆序列表形式

5. **pow_list()**
   - 细节：
     - 幂指数为整型
     - 0 的底数不能为 0（异常处理）
     - 任何非零数的 0 次方为 1
     - 任何数的 1 次方是自己
     - 采用快速幂算法
     - 套用 mul_list() 操作

1. **add()**
   - 逻辑：
     - 同号正常相加
     - 异号先比较数值大小，套用减法得差的绝对值，然后将符号放回

2. **sub()**
   - 逻辑：
     - 分类讨论 str2 的正负
     - 套用 add() 函数实现

3. **mul()**

4. **div()**
   - 异常处理：要求除数非负
   - 输出：元组（取商时采用 div()[0]）

5. **pow()**
   - 异常处理：指数非负
